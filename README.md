# 🤖 Monellyn Broadcast Bot

Бот для рассылки уведомлений мерчантам во все подключённые группы Telegram.

## Команды

| Команда | Описание |
|---|---|
| `/start` | Приветствие и список команд |
| `/myid` | Узнать свой Telegram ID |
| `/groups` | Список всех подключённых групп |
| `/broadcast <текст>` | Разослать сообщение во все группы |

## Деплой на Railway

1. Зайдите на [railway.app](https://railway.app) и войдите через GitHub
2. Нажмите **New Project → Deploy from GitHub repo**
3. Загрузите эти файлы в GitHub репозиторий (или используйте Railway CLI)
4. В настройках проекта добавьте переменные окружения:
   - `BOT_TOKEN` = ваш токен от BotFather
   - `ADMIN_IDS` = ваш Telegram ID через запятую (узнайте через /myid)
5. Railway сам запустит бота!

## Использование

1. Добавьте бота в каждую группу мерчантов как администратора
2. Напишите боту в **личку**: `/broadcast Мы уходим на каникулы с 1 по 5 мая 🎉`
3. Бот разошлёт сообщение во все группы

## Пример сообщений

```
/broadcast 🎉 Уважаемые партнёры, с 1 по 5 мая мы на праздниках. Работа возобновится 6 мая!

/broadcast 🆕 Отличные новости! Теперь доступен новый метод оплаты — криптовалюта USDT TRC-20.

/broadcast ⚠️ Технические работы сегодня с 22:00 до 00:00. Возможны кратковременные перебои.
```
